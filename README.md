# Pink Point - Track your climbing progress

# Building the app

1. Install npm packages

   ```shell
   npm install
   ```

2. Install bower packages

   ```shell
   bower install
   ```

3. Build App

   ```shell
   gulp build
   ```

# Develop the app

1. Start dev server

   ```shell
   gulp watch
   ```

# Running the app

1. Add platform

   ```shell
   ionic platform add ios
   ```

2. Run App

   ```shell
   ionic emulate ios
   ```
